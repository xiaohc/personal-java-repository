# 个人Java库

## 简介
个人Java库，用于快速搭建项目，内容主要是收集的范例，整理的日常经验。

## 组织结构

``` lua
personal-java-repository
├── commons -- 工具类及通用代码模块
├── commons-dependencies -- 通用模块三方件的依赖版本统一管理
├── demo -- 演示项目
│   ├── demo-consumer -- 消费者演示项目
├── demo-dependencies -- 演示项目二方件的依赖版本统一管理
└── demo-parent -- 演示项目统一的Maven父类
```
